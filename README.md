# Graphene Task Hub 🚀

A full-stack task management application built as part of the Studio Graphene Associate Software Engineer assessment.

## Features

* Create Tasks
* Edit Tasks
* Delete Tasks
* Search Tasks
* Filter Tasks (All / Active / Completed)
* Due Date Support
* Task Status Tracking
* Overdue Task Detection
* Dashboard Statistics
* Responsive UI
* Persistent Data Storage using JSON

## Tech Stack

### Frontend

* React
* Vite
* Axios

### Backend

* Node.js
* Express.js

## Project Structure

Graphene-Task-Manager

├── client (React Frontend)

└── server (Node.js Backend)

## Installation

### Backend

cd server

npm install

npm run dev

### Frontend

cd client

npm install

npm run dev
